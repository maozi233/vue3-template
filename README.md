# vue3-template

## Project setup
```
// 如果安装后打开白屏, 用管理员模式打开控制台，执行set-ExecutionPolicy RemoteSigned
pnpm install
```

### Compiles and hot-reloads for development
```
pnpm run serve
```

### Compiles and minifies for production
```
pnpm run build
```

### Lints and fixes files
```
pnpm run lint
```
