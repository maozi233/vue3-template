# vue3-template


## Project setup
```
建议设置淘宝镜像 npm config set registry https://registry.npm.taobao.org
如果用pnpm安装后打开白屏, 用管理员模式打开控制台，执行set-ExecutionPolicy RemoteSigned
npm install
```
